WordPress Username Finder Tool

Find publicly available WordPress usernames from any website

How this tool works

This tool uses the official WordPress REST API endpoint /wp-json/wp/v2/users/ to detect publicly accessible usernames.

Is this legal?

Yes. The tool only reads publicly available information exposed by the website.
